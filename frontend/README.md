# Onshore Frontend (JS/JSX)

React + Vite frontend for the Onshore reputation management assignment.

## What it does

- lets you search/create a business
- opens a dashboard with reputation overview
- shows filtered reviews by sentiment
- lists AI-style action recommendations

## Environment

Create `.env` (optional):

```
VITE_API_BASE_URL=http://localhost:4000/api
```

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
